## 一、说明

- **导航频道名**：电视APK导航栏上显示的频道名称
- **instanceId**：Charles 中 GuidePlusList 返回的 `bindInstanceId`，同时也是 Map Local 规则中 `GetInitMetaData?instanceId=xxx` 的参数值
- **测试桩文件**：Charles 用 Map Local 拦截 GetInitMetaData 请求后返回的本地 JSON 文件路径
- **模板ID**：文档中的 templateId，对应 GetInitMetaData JSON 中 datas[].defaultTemlpateData.templateId
- **操作方式**：打开 APK → 导航栏切换到对应频道 → 该频道返回的 JSON 即包含要测的通栏模板
